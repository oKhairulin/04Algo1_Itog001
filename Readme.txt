=== Виртуальный Интернет-магазин с корзиной покупок ===

    1. ОПИСАНИЕ ПРОГРАММЫ
Консольное приложение для управления виртуальным Интернет-магазином с функциями:
- Управление каталогом товаров
- Формирование корзины покупок
- 4 алгоритма сортировки товаров
- Система скидок
- Сохранение/загрузка данных

    2. ТРЕБОВАНИЯ
- Python 3.6+
- Без дополнительных зависимостей

    3. ЗАПУСК
1) Запустите: python 04Algo_Itog001.py

    4. ФАЙЛЫ
- 04Algo_Itog001.py - основной код
- catalog.json - данные каталога (создается автоматически)
- Readme.txt - текстовый файл описания

    5. ГЛАВНОЕ МЕНЮ
===== ВИРТУАЛЬНЫЙ ИНТЕРНЕТ-МАГАЗИН =====
 1. Просмотр каталога товаров
 2. Добавить товар в корзину
 3. Удалить товар из корзины
 4. Просмотр корзину
 5. Очистить корзину
 6. Применить скидку
 7. Сортировать корзину
 8. Сохранить каталог в файл
 9. Загрузить каталог из файла
10. Добавить новый товар в каталог
11. Редактировать товар в каталоге
12. Удалить товар из каталога
 0. Выход

    6. АЛГОРИТМЫ СОРТИРОВКИ
- Пузырьковая
- Вставками
- Быстрая
- Слиянием

    7. КРИТЕРИИ СОРТИРОВКИ
- По цене
- По весу
- По категории

    8. ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
- Добавление: выберите товар из каталога, укажите количество
- Скидки: введите % скидки (например 15)
- Сортировка: выберите алгоритм, критерий и порядок

    9. ОСОБЕННОСТИ
- Автоматическая загрузка начальных данных товаров
- Предпросмотр при сортировке
- Проверка вводимых данных

Для подробной инструкции по каждой функции запустите программу и следуйте подсказкам.
